This is a sample project that demonstates how to make user extensions for Serious Sam Classics Patch.
This folder contains project files for different build environments.

"Source"
Sample code that contains descriptions for every plugin event and some usage examples.

"PluginModule.vcxproj", "PluginModule.vcxproj.filters" and "PluginModule.vcxproj.user"
Project files for the global solution that builds the entirety of Serious Sam Classics Patch using Visual Studio 2010 or newer.

"PluginModule.dsw" and "PluginModule.dsp"
Standalone project files for Microsoft Visual C++ 6.0. Output and intermediate directories are "../../Bin/" and "../../Obj/" respectively (the same as the new project files).
